undermore / mo-dash

All the stuff that should already be there, but isn't

Undermore collects a set of standard JavaScript utilities, exposed as underscore/lo-dash mixins (so you can keep your utilities in the _.* namespace. It also provides an optional es6-shim and hopeful future JS String prototype methods as well as some helpful jquery extensions that are too minimal to be full-fledge plugins but have become a standard addition to the library.
